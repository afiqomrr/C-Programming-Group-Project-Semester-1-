# Library Book Management System (LBMS)

The **Library Book Management System (LBMS)** is a console-based application developed using the **C programming language** to assist librarians in efficiently managing daily library operations.

The system provides functionalities to:

- 📚 Manage and maintain book records
- 🔍 Monitor book availability and borrowing status
- 📝 Handle borrowing and returning transactions
- 💾 Store and preserve transactional data across application sessions using file-based storage

LBMS is designed to provide a simple, efficient, and user-friendly text-based interface for basic library management tasks.
